# AI CI/CD Failure Analyzer

This project is an Agentic AI-based CI/CD log analyzer built using Streamlit, Groq API, and GitHub Actions.

## Features
- Upload CI/CD logs
- AI-based error analysis
- Root cause detection
- Fix suggestion
- GitHub Actions CI pipeline

## Tech Stack
- Python
- Streamlit
- Groq
- GitHub Actions

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
